## 二、T+1交易制度

### 核心规则

```
当日买入 → 下一交易日才能卖出
当日卖出 → 当日可以买入（卖出资金立即可用）
```

### 实操影响

| 场景 | 操作 | 结果 |
|------|------|------|
| 今日买入 | 不可卖出 | 持仓锁定 |
| 今日卖出 | 可买入 | 新建另一仓位 |
| 昨日持仓 | 今日可卖出 | 自由操作 |

### 对散户的影响

1. **不能当日止损** — 买入后当天不能卖，只能等到明天
2. **资金效率降低** — 卖出后资金需T+1才能转出
3. **避免追高** — 买入后下跌只能等待，无法及时止损

## 三、涨跌停板制度

### 涨跌幅度

| 股票类型 | 涨跌幅限制 | 说明 |
|----------|------------|------|
| **普通股票** | ±10% | 大部分A股 |
| **ST股票** | ±5% | 连续亏损被特别处理 |
| **\*ST股票** | ±5% | 退市风险警示 |
| **新股首日** | ±44% | 上市首日无限制 |
| **科创板/创业板** | ±20% | 注册制股票 |

### 涨跌停日K线特征

```
涨停日K线:
   ┌────┐
   │    │ ← 最高=收盘=涨停价
   │    │
   │    │ ← 可能有下影线
   └────┘
   开===收（涨停价）

跌停日K线:
   ┌────┐
   │    │ ← 可能有上影线
   │    │
   │    │
   └────┘
   开===收（跌停价）
```

### 涨跌停日的成交量特征

| 类型 | 成交量特征 | 含义 |
|------|------------|------|
| **缩量涨停** | 量能<前日 | 主力控盘，锁仓拉升 |
| **放量涨停** | 量能>前日 | 多方力量强，可能开板 |
| **缩量跌停** | 量能<前日 | 主力未出逃 |
| **放量跌停** | 量能>前日 | 恐慌抛售，可能继续跌 |

## 四、集合竞价机制

### 成交原则

```
最大成交量原则:
1. 成交量最大的价格确定为开盘价
2. 高于开盘价的买入申报 → 全部成交
3. 低于开盘价的卖出申报 → 全部成交
4. 等于开盘价的申报 → 根据申报时间先后成交
```

### 集合竞价识别的技巧

| 现象 | 解读 | 操作建议 |
|------|------|----------|
| 高开5%+ | 多方强势 | 观察是否有题材支撑 |
| 低开5%+ | 空方强势 | 谨慎，警惕恐慌抛售 |
| 平开 | 多空平衡 | 等待方向选择 |
| 涨停价开盘 | 一字板 | 次日高开预期 |
| 跌停价开盘 | 一字板 | 次日低开预期 |

## 九、关键提醒

⚠️ **T+1限制**：当日买入不能当日卖出，避免追高被套

⚠️ **涨跌停流动性**：一字板股票无法买入，排队也买不到

⚠️ **尾盘异动**：14:30后的大单往往预示次日方向

⚠️ **集合竞价申报**：9:20前可撤单，9:20-9:25不可撤单
