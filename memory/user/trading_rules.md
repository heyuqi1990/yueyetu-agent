## 一、交易时间（官方标准）

| 时段 | 时间 | 性质 | 规则 |
|------|------|------|------|
| 早盘集合竞价 | 9:15-9:20 | 可挂可撤 | 主力常利用这段时间挂假单试探 |
| 早盘集合竞价 | 9:20-9:25 | 只挂不撤 | 真实报价，反映开盘意图 |
| 静默期 | 9:25-9:30 | 不接受申报 | 撮合系统休息 |
| 正常交易时间 | 9:30-11:30 | 逐笔撮合 | 实时成交 |
| 正常交易时间 | 13:00-14:57 | 逐笔撮合 | 实时成交 |
| 尾盘集合竞价 | 14:57-15:00 | 只挂不撤 | 参考9:20-9:25规则，不可撤单 |

**注意**：9:30-11:30和13:00-14:57这两个时间段叫**正常交易时间**（非连续竞价），逐笔撮合成交。

## 二、竞价成交原则

**价格优先、时间优先**
- 较高买价优先成交
- 较低卖价优先成交
- 同价格下，先挂单先成交

## 三、申报规则

| 规则 | 说明 |
|------|------|
| 最小单位 | 1手 = 100股 |
| 买入 | 必须整手（100股整数倍） |
| 卖出 | 不足100股的碎股可一次性申报卖出 |
| 价格最小变动 | 0.01元 |
| 价格笼子 | 超出基准价±2%（主板）的限价申报作废 |

## 四、T+1 交易制度（核心！）

- **当日买入**股票，下一交易日（T+1）才能卖出
- 卖出资金**当日可用**（可买其他股票）
- **转出银行卡需T+1**
- 银证转账时间：交易日9:00-16:00

**实战影响**：
- 短线买入后当天无法止损出局
- 周五卖出，资金周一才能转出
- 需提前规划资金周转

**临时停牌**（无涨跌幅新股）：股价较开盘价涨跌达到30%、60%时，各停牌10分钟。

## 十、用户交易实战习惯

### 龙头强势股买入逻辑（核心）
用户（鑫鑫淼）短线交易核心逻辑：

**买入时机**：集合竞价阶段
1. **9:15-9:20**：观察板块内个股竞价变化，可撤单阶段
2. **9:20-9:25**：重点关注！真实竞价阶段，不可撤单
   - 重点看**同板块所有个股**的集合竞价情况
   - 通过板块整体竞价情绪判断当日主线方向
3. **9:25-9:30**：静默期（不接受申报）
   - 根据9:15-9:25的竞价推断，选择买入股票
   - 这是开盘前最后的决策窗口

**关键**：通过同板块个股横向竞价对比判断情绪强度，强势股/龙头在竞价阶段就会表现出明显优势

### 容量核心股识别（龙头股策略补充）

**触发条件**：当一个新版块起爆时

**操作步骤**：
1. 第一时间关注该板块里最强的股票
2. 如果板块走出延续性（持续上涨）
3. 识别板块内"屹立不倒"的股票 → 这就是**容量核心股**

**容量核心股特征**：
- 不一定是高度最高的
- 不一定是上板速度最快的
- 但一定是**向上趋势一直在走**的
- 并且是**走的最远的**

**本质**：容量核心股是板块内最终走出来的那只，不是瞬间最耀眼的，而是持续性最强的

**用途**：追踪容量核心股的后续动态，等待超跌反弹机会

### AI任务：龙头股每日追踪推送（独立系统）

**注意**：龙头股系统和超跌反弹系统是**两个独立体系**，分开追踪

**追踪目标**：
1. 每日识别**最强股**（打高度的龙头）
2. 识别**容量核心股**（延续性强的个股）
3. 当两者**合并**时推送该个股
4. 识别板块内**最有带动性**的个股

**推送条件**：
- 新版块起爆时 → 立即推送最强股
- 板块走出延续性 → 推送容量核心股
- 最强股 = 容量核心股（合并）→ 推送合并后的那只
- 每日定期推送龙头股池动态

**独立系统**：
- 龙头股系统：追强势、追龙头（顺势）
- 超跌反弹系统：等超跌、等企稳（逆势）

---

### 最强股与容量核心股的关系（4种情况）

| 情况 | 最强股 | 容量核心股 | 说明 |
|------|--------|------------|------|
| 1 | 合并 | 合并 | 最强股 = 容量核心股（同一只） |
| 2 | 分开 | 分开 | 最强股打高度，容量核心股在延续性中呈现 |
| 3 | 带动 | 被带动 | 容量核心股带动整个板块（是发动机） |
| 4 | - | - | 记住板块内**最有带动性**的那只股 |

**关键区分**：
- **最强股** → 打高度、吸引眼球
- **容量核心股** → 走持续性、带动板块
- **最有带动性的股** → 全局关键，是板块发动机

**重要**：三者关系是**动态的**，可能是同一只，也可能是三只不同股票

### 超跌反弹买入逻辑

**标的**：前期强势股（连板龙头 or 强趋势股）

**杀跌特征**：
- 经过A杀或狠杀
- 下跌过程中**必须有跌停**（强势股补跌的标志）

**买入时机**：个股企稳 + 板块强势 + 资金流入 → 三者共振时买入
- 个股出现止跌/企稳信号
- 所属板块整体强势
- 板块有资金流入
- 资金开始流入的瞬间 → 触发买入

**本质**：强势股超涨后资金出逃→急跌→情绪释放完毕→企稳→二波机会

**与龙头接力的区别**：
- 龙头接力：涨时追（顺势）
- 超跌反弹：跌完后抄（逆势但预判企稳）

### AI任务：超跌反弹股每日追踪推送

**工作流程**：
1. 每天扫描强势股（连板龙头、强趋势股）
2. 识别A杀/狠杀股票，加入关注池
3. 持续跟踪：个股是否企稳 + 板块是否强势 + 资金是否流入
4. 三者共振时 **第一时间推送** 给用户

**推送条件**：企稳信号 + 板块强势 + 资金流入 三者同时出现

## 十二、交易日历（补充）

- A股交易日：**周一至周五**
- 周末（周六、周日）不交易
- 法定节假日不交易
- 非交易日无法获取实时行情数据

**关键时间窗口**：
- 盘中 9:30-14:57：可获取实时数据（涨停板、历史K线等）
- 15:00 收盘后：立即存档当日数据
- 9:15-9:25 集合竞价：重点关注9:20-9:25的真实竞价
- 14:57-15:00 尾盘竞价：不可撤单
